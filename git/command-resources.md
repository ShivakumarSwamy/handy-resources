### override author for next commits
```shell
git commit --author="ShivakumarSwamy <shivakumarswamy.bg@gmail.com>"
```

### amend author of last commit
```shell
git commit --amend --author="ShivakumarSwamy <shivakumarswamy.bg@gmail.com>"
```
